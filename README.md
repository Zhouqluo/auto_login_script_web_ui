# Auto Login Config Tool

作为后端开发，我每天都要登录很多内部系统（Nacos、后台、测试环境等）。
但 Mac 每次自动填充都要验证，指纹对于手汗用户也不好用🙅。
天天重复输入相同账号密码很烦，于是我写了个自动登录的油猴脚本。

为了让脚本配置更简单，我又做了一个 **Web UI**：不用写 JSON，填写一些信息就能生成脚本。

---

## 它能做什么？

* 按“分组”管理不同系统的登录规则
* 每个站点可配置用户名、密码、协议勾选、验证码框、登录按钮
* 一键生成 Tampermonkey 脚本
* 支持导入 / 导出配置
